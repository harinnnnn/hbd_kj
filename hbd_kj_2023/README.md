# 🎂 남자친구 생일 축하 사이트 🎂


- 개발 툴: Next, TypeScript, Emotion
- 배포 툴: Vercel
- 제작 의도: 흔하지 않은 생일 선물을 해주고자 고민하고 있었을 때, 우연히 어떤 분께서 친구 생일 사이트를 만들어주신 것을 보고 영감을 얻어 제작하게 되었습니다. 급하게 만들어 퀄리티가 조금 떨어지지만, 처음으로 넥스트와 타입스크립트 조합으로 개인 프로젝트를 진행하였습니다.
