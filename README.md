# 🎂 남자친구 생일 축하 사이트 🎂


- 개발 툴: Next, TypeScript, Emotion
- 배포 툴: Vercel
- 제작 의도: 흔하지 않은 생일 선물을 해주고자 고민하고 있었을 때, 우연히 어떤 분께서 친구 생일 사이트를 만들어주신 것을 보고 영감을 얻어 제작하게 되었습니다.
(본 사이트에 남자친구의 사진이 여러 장 올라가 있는 관계로, 사이트 링크 대신 화면의 캡쳐 사진을 첨부하는 점 양해 부탁 드립니다.😅)
